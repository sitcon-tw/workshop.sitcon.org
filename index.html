學生計算機年會

理念

技術力就是你的超能力！除了豐富的年會議程，SITCON 的夥伴們也與各校的社群成員合辦了不少工作坊；從初心者到大法師，SITCON Workshop 定能讓你收獲滿滿，在輕鬆的氣氛中砥礪自己的能力、認識更多社群好友，儲備展翼飛翔的能量！

活動

The Open Source Way 開源之道工作坊 - Coder

“Open Source”? 對資訊人而言既熟悉又陌生的「開放原始碼」，到底是怎麼一回事？無論是初次接觸 open source 的朋友，或是希望更了解開放原始碼相關技術的夥伴，都歡迎報名「開源之道工作坊」，一起深入淺出的遨遊於開源的自由世界！

本活動係由 OSSF 與 政治大學資訊科學系、教育部軟體人才培育計畫 - 社群網路運算與服務 及 學生計算機年會 SITCON 共同主辦。

SITCON Workshop @ NTUST: Titanium Mobile 進化攻防戰 - Alloy MVC

Titanium Alloy 是 Appcelerator Titanium 的 MVC 框架，使用.js、.tss、.xml 實作；除了提升程式碼的質量，更能快速開發能在 Android 及 iOS 上執行的 App。此次活動以 Titanium Alloy 為出發點，介紹 MVC 架構、Backbone.js、TSS 樣式與 XML 元素；並透過 Titanium Alloy 學習框架製程、樣式開發與樣板規則等各大領域的技巧。

時間 2013 年 10 月 16 日 18:30 - 21:00, 地點 國立台灣科技大學研揚 TR214, 詳細資訊 報名頁面
HackGen Workshop - 歡迎來到 GitHub 新手村（大同場）
